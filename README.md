# Secret Loffer Blog

## 如何使用:

1. 修改`_config.yml`内容
2. 删除`_post`内的示例文章
3. 写新文章

## 如何写文章

### 如何写未加密(明文)文章:

1. 在`_post`文件夹新建一个文件, 命名为'title-1999-02-32.md'(替换相应内容)
2. 按以下模板填充

加密文章模板

```
---
layout: post
title: 标题
date: 2020-02-11
Author: 作者
tags: [用半角逗号分隔多个标签, 标签2,标签3]
---
明文文章内容
```

### 如何写加密文章:

1. 进入`关于`页面
2. 用1-3个密钥加密文本并复制加密后内容
3. 在`_post`文件夹新建一个文件, 命名为'title-1999-02-32.md'(替换相应内容)
4. 按以下模板填充

加密文章模板

```
---
layout: post
title: 标题
date: 2020-02-11
Author: 作者
tags: [用半角逗号分隔多个标签, 标签2,标签3]
AES: 加密后内容,如:U2FsdGVkX18ZJnLdbRJve78jySoGseY3M3M3f3IIS1I=
---
明文内容,显示效果同未加密文章
```
